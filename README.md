## 周易易经64卦详解

### 本地运行

```bash
python3 -m venv .venv
source ./.venv/bin/activate
pip3 install -r requirements.txt
sudo ufw allow 8888/tcp

# mkdocs serve
mkdocs serve -a 0.0.0.0:8888
```

浏览器打开： `http://0.0.0.0:8888`

或者：http://127.0.0.1:8888
