# 千孝文简介

## 作者简介

韩荣华教授：美国普林顿大学客座教授、美国教育学博士、台湾兴国管理学院企管系教授、创作天下第一文【千孝文】，立志弘扬传统文化的启蒙国学教育，捐赠海峡两岸1000所小学【千孝文】，被聘为海峡两岸唯一推行孝文化特使。目前也是国家十二五课题“学习习惯养成”专家学者、北师大学生课堂专家教授、国家发改委五行易理与管理专家、人大、清华、北大MBA课程教授。

韩荣华教授1988年外交官特考合格，获1996年台湾杰出经理人奖;善长管理思想、战略、运营管理、领导激励、管理系统、组织发展、营销团队、经销商营运管理等课程，现服务天下一家基金会、中国企业学习网签专家。

近年来，韩荣华教授致力于在全球范围内人文关怀和慈善精神的传播，韩荣华教授为教育事业、希望工程、支援灾区、弘扬优秀民族文化等各项公益事业贡献杰出，被儒林国学网聘为“首席国学顾问”。

## 千孝文简介

本文的创作实属首创，乃继孔子所著之《孝经》之后，首篇以孝为核心的论著，被誉为“天下第一文启蒙圣书”。全文以《千字文》为冠首，写下的四千字诗文，除“孝”一字重复使用之外，其余的三千字竟无一字重复，囊括了所有常用中文字的百分之九十左右。以《千字文》为冠首所写的诗文，意味着一千首诗文，作者依每四个字为一句的诗文，尝试说明其内涵，探索每句诗文之意义。

## 千孝文【第一百五十七句：】岂只痛孝(qǐ zhǐ tòng xiào)

【释义：】昔日有一汪洋大盗，被判极刑，临行刑时，狠心猛咬其母乳头，怨其母曰：“儿幼小时，如抢得钱物，即赞儿聪明，偷窃他人之物，即赞儿机智，抢劫回家，则赞儿为英雄，一向勉儿行凶作恶，才落得今日下场，全因母未善尽教育有以致之，故咬断乳头以泄心中之恨。”为人父母者若见此故事，宜警惕，子女若有过，要严加管教，不可过于放纵，否则，日后沦为不良少年，岂只痛乳头而已，身心皆会哀痛一辈子的。
